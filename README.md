# iGEM USTC Software 2015 后端代码仓库
## 目录结构
`doc`目录下存放面向软件开发者和使用者的文档
`src`目录下存放的是django源代码，`src\src`下存放软件逻辑代码，`src\test`目录下存放单元测试代码
